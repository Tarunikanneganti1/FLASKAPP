# Resume Classifier using Flask

This project classifies resumes based on their content using a Machine Learning model and Flask.

## Features
- Upload resumes in PDF, DOCX, or TXT format
- Extract text and classify the resume into categories like Java Developer, Data Scientist, etc.

## Technologies Used
- Python, Flask, Scikit-learn, Joblib
- HTML, CSS (for the front-end)

## How to Run
1. Clone the repository: `git clone <your-repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the Flask app: `python app.py
